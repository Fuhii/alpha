# デジタル時計アプリ

このデジタル時計アプリは、年月日、時分秒、曜日を表示します。異なるタイムゾーンで複数の時計を表示でき、夏時間とうるう年を考慮します。デザインテーマを選択することも可能です。

## 特徴

- 異なるタイムゾーンで複数の時計を表示可能
- 夏時間を自動調整
- うるう年を考慮
- デザインのカスタマイズが可能
